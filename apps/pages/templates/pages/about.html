Sobre o Havoc

Sistema moderno de gerenciamento de conteúdo

Nossa Missão

O Havoc é um sistema de gerenciamento de conteúdo desenvolvido com as melhores práticas de desenvolvimento de software, seguindo os princípios SOLID e arquitetura limpa.

Características Principais

  • Arquitetura Limpa: Separação clara de responsabilidades
  • Princípios SOLID: Código maintível e extensível
  • Django Framework: Robustez e segurança
  • Interface Moderna: Bootstrap 5 e design responsivo

Tecnologias Utilizadas

Backend
  • Python 3.12+
  • Django 5.2
  • SQLite/PostgreSQL
Frontend
  • HTML5 & CSS3
  • Bootstrap 5
  • Font Awesome

Estrutura do Projeto

O projeto está organizado em apps especializados:

  • Accounts: Autenticação e gerenciamento de usuários
  • Pages: Sistema de páginas e navegação
  • Config: Configurações e administração
  • Articles: Gerenciamento de artigos (em desenvolvimento)
